## Question 1
Write a program to check if a number is Positive, Negative, or Zero.

The input is an integer. The output should be "Positive", "Negative", or "Zero".

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|10|Positive|
|-5|Negative|
|0|Zero|
|1|Positive|
|-100|Negative|
