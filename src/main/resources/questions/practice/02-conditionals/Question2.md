## Question 2
Write a program to check if a number is Even or Odd.

The input is an integer. The output is "Even" if the number is even, and "Odd" if the number is odd.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|2|Even|
|1|Odd|
|100|Even|
|55|Odd|
|-4|Even|
|0|Even|
