## Question 4
Write a program to check if a number is divisible by both 3 and 5.

The input is an integer. The output should be "Yes" if divisible by both 3 and 5, and "No" otherwise.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|15|Yes|
|30|Yes|
|9|No|
|10|No|
|7|No|
