## Question 5
Write a program to check if a year is a Leap Year.

The input is a year (integer).
Rule: A year is a leap year if it is divisible by 4, EXCEPT if it is divisible by 100 (unless it is also divisible by 400).
Output "Yes" or "No".

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|2020|Yes|
|1900|No|
|2000|Yes|
|2023|No|
