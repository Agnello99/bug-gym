## Question 6
Create a simple calculator using switch-case.

The input consists of two numbers and a character operator (+, -, *, /).
Output the result of the operation. (For division, assume integer division).

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|10 5 +|15|
|10 5 -|5|
|3 4 *|12|
|10 3 /|3|
