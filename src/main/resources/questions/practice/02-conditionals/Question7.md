## Question 7
Check if a character is an Alphabet, Digit, or Symbol.

The input is a single character.
Output "Alphabet" if it is a-z or A-Z.
Output "Digit" if it is 0-9.
Output "Symbol" otherwise.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|A|Alphabet|
|z|Alphabet|
|5|Digit|
|#|Symbol|
