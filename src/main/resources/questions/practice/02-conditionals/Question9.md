## Question 9
Write a program that checks whether a given lowercase letter is a vowel or a consonant.

The input is a single lowercase letter. Output "Vowel" if it is a vowel (a, e, i, o, u), otherwise output "Consonant".

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|a|Vowel|
|b|Consonant|
|u|Vowel|
