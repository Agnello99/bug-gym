## Question 6
Check if a number is a Prime Number.

The input is an integer N (N > 1).
Output "Yes" if it is prime, "No" if it is not.
(A prime number has exactly two factors: 1 and itself).

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|5|Yes|
|4|No|
|13|Yes|
|15|No|
|2|Yes|
