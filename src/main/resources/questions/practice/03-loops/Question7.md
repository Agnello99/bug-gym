## Question 7
Print the first N terms of the Fibonacci sequence.

The input is an integer N.
The sequence starts: 0, 1, 1, 2, 3, 5...
Output the first N numbers separated by spaces.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|5|0 1 1 2 3|
|3|0 1 1|
|1|0|
|7|0 1 1 2 3 5 8|
