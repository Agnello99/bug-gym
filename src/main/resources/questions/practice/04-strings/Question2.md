## Question 2
Check if a string is a palindrome (reads the same forwards and backwards). Case sensitive.

The input is a string. The output is "Yes" if it is a palindrome, "No" otherwise.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|madam|Yes|
|hello|No|
|Racecar|No|
|racecar|Yes|
|aba|Yes|
