## Question 4
Write a program that repeats a given string N times.

The input consists of a string S followed by an integer N. The output is the string concatenated with itself N times.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|Hi 3|HiHiHi|
|A 5|AAAAA|
|Hello 1|Hello|
|Java 0||
