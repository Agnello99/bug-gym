## Question 5
Write a program to remove all spaces from a given string.

The input is a string (which may contain spaces). The output is the same string with all spaces removed.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|Hello World|HelloWorld|
|Java Programming|JavaProgramming|
| A B C |ABC|
|NoSpaces|NoSpaces|
