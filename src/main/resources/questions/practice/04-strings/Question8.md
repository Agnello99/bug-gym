## Question 8
Replace all vowels (a, e, i, o, u) in a string with an asterisk (*). Case insensitive.

The input is a string. The output is the modified string.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|Hello|H\*ll\*|
|Apple|\*ppl\*|
|Java|J\*v\*|
|Rhythm|Rhythm|
