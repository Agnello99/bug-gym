## Question 10
Find the LAST index of a specific element in an array.

The input starts with N (size). Then N integers. Finally, a Target integer.
Output the last index (0-based) where Target is found. If not found, output -1.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|5 <br> 1 2 3 2 1 <br> 2|3|
|5 <br> 1 2 3 2 1 <br> 1|4|
|3 <br> 1 1 1 <br> 1|2|
|3 <br> 1 2 3 <br> 5|-1|
