## Question 11
Calculate the sum of all elements in a 1D array.

The input starts with integers N (size), followed by N integers. The output is the total sum.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|5 <br> 1 2 3 4 5|15|
|3 <br> 10 -5 5|10|
|1 <br> 100|100|
