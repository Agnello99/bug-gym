## Question 12
Check if an array contains any duplicate elements.
Input starts with N (size), followed by N integers. Output "Yes" if duplicates exist, "No" otherwise.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|5 <br> 1 2 3 4 5|No|
|4 <br> 1 2 3 1|Yes|
|3 <br> 5 5 5|Yes|
|1 <br> 10|No|
