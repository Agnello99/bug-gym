## Question 14
Move all zeros in an array to the end, while maintaining the relative order of non-zero elements.
Input starts with N (size), followed by N integers. Output the modified array.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|5 <br> 0 1 0 3 12|1 3 12 0 0|
|3 <br> 0 0 1|1 0 0|
|4 <br> 1 2 3 4|1 2 3 4|
|2 <br> 0 0|0 0|
