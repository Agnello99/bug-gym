## Question 15
Write a program that finds the second largest distinct element in an array.

The input starts with an integer N, followed by N integers. The output is the second largest distinct value.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|5 <br> 1 4 2 9 7|7|
|4 <br> 10 10 8 6|8|
