## Question 16
Write a program that inserts an element into an array at a given position.

The input starts with an integer N, followed by N integers, then an integer X (value to insert) and an integer P (1-based position). The output is the new array after insertion, space-separated.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|4 <br> 1 2 3 4 <br> 9 2|1 9 2 3 4|
|3 <br> 5 6 7 <br> 8 4|5 6 7 8|
