## Question 2
Find the average of N floating point numbers.

The input starts with N (integer), followed by N decimal numbers. The output is the average, printed to 2 decimal places (use String.format or System.out.printf).

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|2 <br> 10.0 20.0|15.00|
|3 <br> 1.0 2.0 3.0|2.00|
|1 <br> 5.5|5.50|
