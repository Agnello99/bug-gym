## Question 5
Find the minimum element in an array of integers.

The input starts with an integer N (size of array), followed by N integers. The output is the minimum value among them.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|5 <br> 1 2 3 4 5|1|
|3 <br> 10 20 5|5|
|4 <br> -1 -5 -2 -3|-5|
|1 <br> 0|0|
|5 <br> 100 200 500 300 400|100|
