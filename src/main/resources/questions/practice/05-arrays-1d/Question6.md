## Question 6
Write a program to reverse an array of integers.

The input starts with an integer N (size), followed by N integers. The output is the array elements in reverse order.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|5 <br> 1 2 3 4 5|5 4 3 2 1|
|3 <br> 10 20 30|30 20 10|
|1 <br> 7|7|
