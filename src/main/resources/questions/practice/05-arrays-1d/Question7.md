## Question 7
Write a program to swap the first and last elements of an array.

The input starts with an integer N (size, N >= 1), followed by N integers. The output is the array with the first and last elements swapped.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|5 <br> 1 2 3 4 5|5 2 3 4 1|
|3 <br> 10 20 30|30 20 10|
|4 <br> 5 0 0 2|2 0 0 5|
|1 <br> 99|99|
