## Question 8
Check if an array of integers is sorted in ascending order.

The input starts with an integer N (size), followed by N integers. The output is "Yes" if strictly increasing or equal, "No" otherwise.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|5 <br> 1 2 3 4 5|Yes|
|3 <br> 1 3 2|No|
|4 <br> 10 10 10 10|Yes|
|3 <br> 5 1 2|No|
