## Question 9
Calculate the element-wise sum of two arrays.

The input starts with N (size). Then N integers for Array 1, followed by N integers for Array 2.
Output the resulting array where Index[i] = Arr1[i] + Arr2[i].

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|3 <br> 1 2 3 <br> 4 5 6|5 7 9|
|2 <br> 10 20 <br> -10 -20|0 0|
|3 <br> 1 1 1 <br> 2 2 2|3 3 3|
