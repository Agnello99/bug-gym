## Question 1
Calculate the sum of all elements in a 2D matrix.

The input starts with two integers R (rows) and C (columns). Then R*C integers follow. output the total sum.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|2 2 <br> 1 2 <br> 3 4|10|
|1 3 <br> 5 5 5|15|
|3 1 <br> 1 <br> 2 <br> 3|6|
|2 2 <br> -1 -1 <br> -1 -1|-4|
