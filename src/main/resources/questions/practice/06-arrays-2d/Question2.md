## Question 2
Write a program to add two matrices of the same size.

The input starts with two integers R (Rows) and C (Columns).
Then R*C integers for the first matrix.
Then R*C integers for the second matrix.
The output is the resulting matrix elements (row by row).

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|2 2 <br> 1 2 3 4 <br> 5 6 7 8|6 8 10 12|
|2 3 <br> 1 1 1 1 1 1 <br> 2 2 2 2 2 2|3 3 3 3 3 3|
|1 1 <br> 10 <br> 20|30|
