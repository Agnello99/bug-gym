## Question 3
Calculate the sum of each row in a 2D matrix.

The input starts with R (Rows) and C (Columns), followed by the matrix elements.
The output should be R integers, where each integer is the sum of a row.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|2 3 <br> 1 2 3 <br> 4 5 6|6 15|
|3 2 <br> 1 1 <br> 2 2 <br> 3 3|2 4 6|
|2 2 <br> 10 20 <br> -5 -5|30 -10|
