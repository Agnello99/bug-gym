## Question 4
Find the sum of the main diagonal elements of a square matrix.
(The main diagonal consists of elements where row index equals column index: [0][0], [1][1], etc.)

The input starts with an integer N (representing an N x N matrix), followed by N*N integers.
The output is the sum of diagonal elements.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|3 <br> 1 2 3 <br> 4 5 6 <br> 7 8 9|15|
|2 <br> 10 20 <br> 30 40|50|
|1 <br> 5|5|
