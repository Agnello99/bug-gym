## Question 5
Find the maximum element in a 2D matrix.

The input starts with R (Rows) and C (Columns), followed by R*C elements.
The output is the maximum value found in the matrix.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|2 3 <br> 1 2 9 <br> 4 5 6|9|
|2 2 <br> -1 -5 <br> -2 -10|-1|
|3 3 <br> 0 0 0 <br> 0 1 0 <br> 0 0 0|1|
