## Question 8
Check if a square matrix is an Identity Matrix.
(Ones on the main diagonal, zeros everywhere else).

The input starts with an integer N (size N x N), followed by N*N elements.
Output "Yes" if it is an identity matrix, "No" otherwise.

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|2 <br> 1 0 <br> 0 1|Yes|
|2 <br> 1 2 <br> 3 4|No|
|3 <br> 1 0 0 <br> 0 1 0 <br> 0 0 1|Yes|
|3 <br> 1 0 0 <br> 0 0 0 <br> 0 0 1|No|
