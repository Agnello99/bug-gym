## Question 9
Write a program to check whether a square matrix is symmetric.

The input starts with an integer N, followed by an N x N matrix. Output "Symmetric" if the matrix equals its transpose, otherwise output "Not Symmetric".

### For example:
| **Input** | **Result** |
|:--------- |:-----------|
|3 <br> 1 2 3 <br> 2 4 5 <br> 3 5 6|Symmetric|
|2 <br> 1 0 <br> 2 1|Not Symmetric|
